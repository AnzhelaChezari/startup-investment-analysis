# Исследование рынка стартапов и стратегий финансирования 

Проект посвящен анализу инвестиционной привлекательности стартапов, сегментации рынка по типам финансирования и выявлению рыночных трендов за 2014 год.

## Технологический стек
*   **Язык:** Python (Pandas, NumPy).
*   **Визуализация:** Matplotlib, Seaborn (анализ распределений, динамика роста).
*   **Анализ:** Статистический анализ выбросов, анализ длительности финансирования.

## Решение задач проекта:

* **1 Задача:** В первой задаче знакомлюсь с данными, обрабатываю типы данных, ищу полные и не явные дубликаты, оцениваю пропуски в столбцах (каждый столбец отношу к одному из трех типов пропуска MAR, MNAR, MCAR). Делаю выводы, что почти во всех таблицах присутствуют пропуски, от 10% до 98%. Оценивая типы пропусков и полноту данных, делаю вывод, что данных достаточно для следующего анализа.

* **2 Задача:** Затем с помощью сводной таблицы изучаю как проводилось финансирование из года в год, отвечаю на вопросы: в каком году типичный размер собранных в рамках одного раунда средств был максимален. а с помощью графиков получаю ответ на вопрос, в каком году был пик финансирования. Заказчику надо было узнать, сколько сотрудников работают с высшим образованием, для этого, был выделен признак (1/0), посчитана доля "без образования" для разных типов компаний ( разделение компаний происходило по заданному условию, применяя функцию cut).

* **3 Задача:** Для заказчика важно определить, какие компании готовы идти на сделку, по заданию это компании, которые получают финансирование. Построив гистограмму распределенения и ящик с усами (boxplot), выявили сильный размах вправо и аномальные выбросы, поэтому для среднего финансирования выбрали медиану значений. Дальше были выявлены самые успешные продажи за весь период, выделены перспективные отрасли для вложений - критерии: топы по медианной цене и среднеквадратичному отклонению.
