# Исследование рынка стартапов и стратегий финансирования 

Проект посвящен анализу инвестиционной привлекательности стартапов, сегментации рынка по типам финансирования и выявлению рыночных трендов за 2014 год.

## Технологический стек
*   **Язык:** Python (Pandas, NumPy).
*   **Визуализация:** Matplotlib, Seaborn (анализ распределений, динамика роста).
*   **Анализ:** Статистический анализ выбросов, анализ длительности финансирования.

## Решение задач проекта:

* **1 Задача:** В первой задаче знакомлюсь с данными, обрабатываю типы данных, ищу полные и не явные дубликаты, оцениваю пропуски в столбцах (каждый столбец отношу к одному из трех типов пропуска MAR, MNAR, MCAR). Делаю выводы, что почти во всех таблицах присутствуют пропуски, от 10% до 98%. Оценивая типы пропусков и полноту данных, делаю вывод, что данных достаточно для следующего анализа.

* **2 Задача:** Затем с помощью сводной таблицы изучаю как проводилось финансирование из года в год, отвечаю на вопросы: в каком году типичный размер собранных в рамках одного раунда средств был максимален. а с помощью графиков получаю ответ на вопрос, в каком году был пик финансирования.
